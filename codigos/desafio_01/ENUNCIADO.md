# Desafio 1
A RTech está desenvolvendo uma API de controle de solicitações internas (como férias, reembolsos e treinamentos).  
Essa API deverá permitir criar, listar, atualizar e excluir solicitações no sistema, armazenando-as em memória (ou banco de dados simples, opcional).  
Sua missão é desenvolver um protótipo funcional dessa API ou escrever o esqueleto de código que demonstra o raciocínio de solução.
