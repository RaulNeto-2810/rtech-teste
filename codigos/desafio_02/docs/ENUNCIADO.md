# Desafio 2

Implemente uma aplicação simples (em Java ou Python) que:

- Cadastre novas solicitações (com campos: id, tipo, descricao, status).
- Liste todas as solicitações.
- Atualize o status de uma solicitação existente.
- Exclua uma solicitação pelo id.

## Requisitos mínimos

- Utilizar boas práticas de código (métodos curtos, nomes claros, comentários quando necessário).
- Armazenar os dados em memória (lista, dicionário ou array).
- Retornar mensagens adequadas para erros (ex.: item não encontrado).
